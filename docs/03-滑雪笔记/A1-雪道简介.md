# 雪道简介

## 雪道评级系统

### 欧洲

| 标记颜色       | 难度等级               | 说明                                                            |
| ------------- | ---------------------- | -------------------------------------------------------------- |
| 🟢 **Green**  | 初学者                 | 很缓、非常容易，多见于雪场底部或练习区（法国、波兰、西班牙、英国等） |
| 🔵 **Blue**   | 容易                   | 一般是宽阔、缓坡，坡度通常不超过25%                               |
| 🔴 **Red**    | 中级                   | 比蓝道更陡、更窄，坡度通常不超过40%                               |
| ⚫️**Black**   | 高级/专家              | 最陡、最难的标记道，可能有moguls（雪包）或不规则地形               |
| 🟠**Orange**  | 超难（部分地区）        | 在奥地利、瑞士等处标识比黑道更难的雪道                            |
| 🟡**Yellow**  | Skiroute/Freerideroute | 标记雪道但不定期压雪，有点介于滑道与野雪之间，需要更高技能          |

### 斯堪的纳维亚（挪威/瑞典/芬兰）

| 颜色 + 形状                  | 难度     | 坡度范围               |
| ---------------------------- | ------- | ---------------------- |
| 🟢 **Green circle**         | 很容易   | 约 ≤ 9° (≈16%)         |
| 🔵 **Blue square**          | 容易     | 约 ≤ 15° (≈27%)        |
| 🟥 **Red rectangle**        | 中级     | 约 ≤ 25° (≈47%)        |
| ♦️ **Black diamond**        | 困难     | > 25° (≈47%)           |
| ⬛ **Double black diamond** | 非常困难 | 极高难度（部分雪场使用） |

### 北美、大洋洲

| 形状 + 颜色                  | 难度级别            | 说明                        |
| ---------------------------- | ------------------ | -------------------------- |
| 🟢 **Green Circle**         | 最容易              | 宽阔且通常压雪，坡度约 6–25% |
| 🔵 **Blue Square**          | 中等                | 中等坡度，通常压雪          |
| ♦️ **Black Diamond**        | 困难                | 高坡度，可能未压雪          |
| ⬛ **Double Black Diamond** | 非常困难            | 更陡更危险                  |
| 🔺 **Triple Black Diamond** | 极端困难（部分雪场） | 极难，仅限专家               |

### 日本

| 颜色           | 难度                     |
| -------------- | ----------------------- |
| 🟢 **Green**  | 初学者（基地或缓坡）      |
| 🔴 **Red**    | 中级滑道（许多滑道为此级） |
| ⬛ **Black**  | 专业/专家                |

## 雪质类别

### 粉雪(Powder Snow)

新鲜降落、未受压的雪，松软、轻盈、干燥，主要出现在新雪刚落及低温环境下。颗粒松散，滑行时会有像“漂浮”的感觉，是滑雪者追求的理想雪况。

### 乱雪/碎雪(Crud Snow)

原本的粉雪被雪板或滑行压碎后形成的一种参差不齐的混合雪，常见于粉雪被大量滑过后形成。表面不平、软硬不一，既有粉雪又有被压实的部分，需要较高技巧应对。

### 硬壳雪(Crust Snow)

雪表层经历融化再重新结冰的循环，在上层形成一层硬而脆的壳，典型在日照融雪后夜间再冻时出现。有时薄得容易破，有时厚到可以直接滑过。

### 压实雪(Packed/Hard Snow)

粉雪或较软雪在雪地机械整理或大量滑行后被压实，常见于人工雪场。表面比较平整、硬度较大，适合刻滑但较滑，速度快。

### 颗粒雪(Corn Snow)

在反复“融化—再冻结”循环下形成的雪质，雪晶逐渐变成均匀、圆润、像玉米粒一样的颗粒。支撑性好，不容易陷板，边刃抓地稳定，非常顺滑，转弯反馈清晰。

### 冰状雪(Ice/Icy Snow)

雪在融化后完全结冰的结果，形成非常坚硬、光滑的表面。滑行时容易打滑，对初学者尤为具有挑战性，常用于经济高山滑雪赛道。

### 湿雪/泥雪(Slush/Sticky Snow)

雪温接近或超过融点后开始部分融化，含水量高、黏稠、沉重，春季或气温较高时较常见。滑行阻力大、速度慢，容易“粘住”滑雪板。
