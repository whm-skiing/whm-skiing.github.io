# 雪道坡度

### 雪道评级系统

##### 欧洲

| 标记颜色       | 难度等级              | 说明                                                            |
| ------------- | --------------------- | -------------------------------------------------------------- |
| 🟢 **Green**  | 初学者                | 很缓、非常容易，多见于雪场底部或练习区（法国、波兰、西班牙、英国等） |
| 🔵 **Blue**   | 容易                  | 一般是宽阔、缓坡，坡度通常不超过 ~25%                             |
| 🔴 **Red**    | 中级                  | 比蓝道更陡、更窄，坡度通常不超过 ~40%                             |
| ⚫️**Black**  | 高级/专家              | 最陡、最难的标记道，可能有moguls（雪包）或不规则地形               |
| 🟠**Orange** | 超难（部分地区）        | 在奥地利、瑞士等处标识比黑道更难的雪道                            |
| 🟡**Yellow** | Skiroute/Freerideroute | 标记雪道但不定期压雪，有点介于滑道与野雪之间，需要更高技能          |


##### 斯堪的纳维亚（挪威/瑞典/芬兰）

| 颜色 + 形状                  | 难度   | 坡度范围        |
| --------------------------- | ---- | ----------------- |
| 🟢 **Green circle**        | 很容易  | 约 ≤ 9° (≈16%)  |
| 🔵 **Blue square**         | 容易   | 约 ≤ 15° (≈27%) |
| 🟥 **Red rectangle**       | 中级   | 约 ≤ 25° (≈47%) |
| ♦️ **Black diamond**       | 困难   | > 25° (≈47%)   |
| ⬛ **Double black diamond** | 非常困难 | 极高难度（部分雪场使用）   |


##### 北美、大洋洲

| 形状 + 颜色                     | 难度级别       | 说明                           |
| --------------------------- | ---------- | ---------------------------- |
| 🟢 **Green Circle**         | 最容易        | 宽阔且通常压雪，坡度约 6–25% |
| 🔵 **Blue Square**          | 中等         | 中等坡度，通常压雪         |
| ♦️ **Black Diamond**        | 困难         | 高坡度，可能未压雪         |
| ⬛ **Double Black Diamond**  | 非常困难       | 更陡更危险             |
| 🔺 **Triple Black Diamond** | 极端困难（部分雪场） | 极难，仅限专家           |

##### 日本

| 颜色           | 难度            |
| ------------ | ------------- |
| 🟢 **Green** | 初学者（基地或缓坡）    |
| 🔴 **Red**   | 中级滑道（许多滑道为此级） |
| ⬛ **Black**  | 专业/专家         |
